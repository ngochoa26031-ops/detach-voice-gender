# Voice DNA

Thu muc nay la nguon duy nhat de gan nhan Nam/Nu. Neu mot speaker khong khop
du chac voi bat ky DNA nao, ket qua phai la `Khong ro`.

Moi DNA gom mot file WAV 16 kHz mono va mot entry trong `library.json`:

- `name`: ten DNA hien trong file speaker.
- `gender`: `male` hoac `female`.
- `file`: file WAV enroll da duoc verify.

Co the bo sung DNA moi ma khong can sua code: them WAV va them entry vao JSON.
